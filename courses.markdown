This is a list of all the technical courses I've taken both in my undergrad at Berkeley and in my master's at UChicago. I plan on updating this list with my review of each class and notes from each of them!


## **UChicago**

### Fall 2025

- **DATA 37711 - Foundations of Machine Learning and AI I (PhD)**
    - Statistical learning, representation learning, causal machine learning, algorithmic fairness
- **STAT 37792 - Topics in Deep Learning: Generative Models**
    - Likelihood models, VAEs, flow and energy-based models, GANs, diffusion models
- **<a href="https://agents4science.github.io/curriculum.html">CMSC 35370</a> - AI Agents for Science**
    - LLMs, RAG, tool calling and MCP, high-performance computing, evaluation and benchmarking

## **UC Berkeley**

### Spring 2025

- **<a href="https://forecastingclass.com/sp25/calendar/">STAT 165</a> - Forecasting**
    - Calibration, scoring rules, Fermi estimation, base rates, reference classes, combining forecasts, mixture models, cognitive biases, Martingales, Kelly betting
- **<a href="https://people.ischool.berkeley.edu/~dbamman/nlp25.html">INFO 159</a> - Natural Language Processing**
    - Word embeddings, attention and transformers, neural language models, LLMs, RAG, agents, NER, information extraction, machine translation

### Fall 2024

- **<a href="https://data102.org/fa24/">DATA 102</a> - Data, Inference, and Decisions**
    - Binary decision-making, decision theory, multiple testing, frequentist and Bayesian inference, uncertainty quantification, GLMs, nonparametric methods, causal inference, concentration inequalities, bandits and reinforcement learning, differential privacy
- **<a href="https://stat153.berkeley.edu/fall-2024/">STAT 153</a> - Introduction to Time Series**
    - Sequential data properties, measures of dependence, regression, regularization and smoothing, spectral analysis, ARIMA, ETS models, Prophet, LSTMs, Calibration
- **<a href="https://datac182fa24.github.io/">DATA 182</a> - Designing, Visualizing and Understanding Deep Neural Networks**
    - Binary classification and logistic regression, optimization, PyTorch, feed-forward networks, CNNs, RNNs, LSTMs, transformers, NLP and pretraining, autoencoders, GPT, GPUs

### Spring 2024

- **<a href="https://people.eecs.berkeley.edu/~jrs/189s24/">CS 189</a> - Introduction to Machine Learning**
    - Training and testing, overfitting and underfitting, gradient descent, SVMs, Bayes risk, Gaussian discriminant analysis, least-squares regression and regularization, logistic regression, bias-variance tradeoff, decision trees and random forests, kernels, feed-forward networks, CNNs, SVD/PCA, clustering, high-dimensional geometry, K-nearest neighbors
- **<a href="https://arxiv.org/pdf/2401.00649">STAT 230A</a> - Linear Models**
    - OLS, Gauss-Markov Theorem, asymptotic inference, sandwich covariance, FWL, model misspecification, regularization, WLS, GLMs, quantile regression, survival analysis

### Fall 2023

- **<a href="https://inst.eecs.berkeley.edu/~ee127/sp23/assets/notes/eecs127_reader.pdf">EECS 127</a> - Optimization Models for Engineering**
    - Vector spaces, eigenvalues, spectral theorem, SVD/PCA, Lp-norms, vector calculus, least-squares, convexity, gradient descent, duality, LPs & QPs, regularization
- **<a href="https://arxiv.org/pdf/2305.18793">STAT 156</a> - Causal Inference**
    - Potential outcomes, randomization, stratification, matching, observational studies, propensity score, confounding and bias, sensitivity analysis, IV/RDD, DAGs
- **<a href="https://arxiv.org/pdf/2305.18793">DATA 101</a> - Data Engineering**
    - Advanced SQL, query optimization, serialization, entity resolution, MongoDB, concurrency, parallel/distributed computing, MapReduce, OLAP


### Spring 2023

- **STAT 135 - Concepts of Statistics**
    - Parameter estimation (MoM, MLE), asymptotic properties, Neyman-Pearson Lemma, hypothesis testing (t-test, Chi-squared test, ANOVA), regression, Bayesian approach (MAP)
- **ECON 140 - Econometrics**
    - Regression analysis, economic statistics, RCTs, instrumental variables, difference-in-differences, regression discontinuity design, errors and confounding

### Fall 2022

- **<a href="https://ds100.org/fa22/">DATA 100</a> - Principles and Techniques of Data Science**
    - Data science ecosystem (pandas, numpy, matplotlib, sklearn, etc.), data cleaning and EDA, intro to ML (regression, classification, PCA, clustering), SQL
- **<a href="https://inferentialthinking.com/chapters/intro.html">DATA 140</a> - Probability for Data Science**
    - Random variables, expectation and (co)variance, discrete and continuous distributions, Markov Chains, CLT, multivariate normals, simple and multiple regression
- **STAT 33B - Introduction to Advanced Programming in R**
    - R, ggplot, tidyverse

### Spring 2022

- **<a href="https://fa22.datastructur.es/">CS 61B</a> - Data Structures**
    - Java, Git, Object-Oriented Programming, Arrays, Trees, Queues, Hashing, Sorting, Graphs
- **<a href="https://inferentialthinking.com/chapters/intro.html">DATA 8</a> - Foundations of Data Science**
    - Data manipulation, random sampling, descriptive statistics, bootstrapping, permutation tests, confidence intervals, regression, K-nearest neighbors, basic probability

### Fall 2021

- **<a href="https://cs61a.org/">CS 61A</a> - The Structure and Interpretation of Computer Programs**
    - Python, Recursion, Basic Data Structures, Scheme, SQL
- **MATH 54 - Linear Algebra**
    - Vectors and matrices, vector spaces, eigenvalues, least-squares, matrix decompositions, projection, diagonalization, Gram-Schmidt process, ordinary differential equations
